# Pilot Checklist — First Real Factory

Use this checklist when onboarding the first production pilot factory.

## Pre-Invitation

- [ ] Factory application reviewed and approved
- [ ] Pilot factory added in **Pilot Program** (status: Invited)
- [ ] Pricing plan selected (Starter / Growth / Trial)
- [ ] Launch Readiness score reviewed (target: ≥ 70)

## Onboarding (status: Onboarding)

- [ ] Tenant created with owner user
- [ ] Client linked to tenant
- [ ] Subscription activated
- [ ] Factory profile completed (company, products, certificates)
- [ ] Export markets configured
- [ ] Demo data cleared if not needed
- [ ] Onboarding wizard completed by factory owner
- [ ] Telegram or primary channel configured (if applicable)

## Active Pilot (status: Active)

- [ ] First content published
- [ ] First lead created
- [ ] First buyer opportunity claimed
- [ ] Weekly check-in scheduled with factory owner
- [ ] Pilot Success dashboard reviewed weekly
- [ ] No critical errors in Error Tracking

## Feedback Phase (status: Feedback Phase)

- [ ] Factory submits feedback via Feedback Center
- [ ] Success score assigned (0–100)
- [ ] ROI discussion with factory leadership
- [ ] Outstanding bugs triaged

## Completion (status: Completed)

- [ ] Pilot retrospective documented in Pilot Program notes
- [ ] Commercial terms confirmed
- [ ] Tenant moved to standard support tier
- [ ] Launch blockers from Launch Readiness resolved
